## 대회 개요

#### 주최&주관
주최 : 과학기술정보통신부 / 주관: 정보통신산업진흥원

#### 목적
- 경진대회를 통해 우수한 기술력을 보유한 인공지능 중소 벤처 기업 발굴하여 사업화를 지원함으로써 인공지능 기술의 활용과 확산을 촉진하는 데에 목적이 있음
- 10개 과제 중 성능 상위 4개 팀(총 40팀)에 대한 사업화 지원 평가 기회 제공
- 종합 평가 상위 20팀에 대한 사업화 지원 (사업화 계획 서류, 발표평가, 경진대회 결과 종합)

## 데이터 특징
- 예측이 필요한 사업장 545개 / 각 사업장별로 예측이 필요한 변수 6개 / 예측기간 699일
  ⇒ 약 230만개 값의 예측이 필요함
- 결측치가 30% 이상인 변수를 가진 사업장이 약 80개. 특히 학습 데이터의 종료기간에 결측치가 연속해 있는 경우가 있어 예측에 활용하기 힘든 경우도 다수 존재
<img width="1427" alt="image" src="https://user-images.githubusercontent.com/30097727/227517404-64cda0e1-1748-42f3-bb9d-5d14fdd6d33f.png">
- 각 시계열 데이터(변수)별 선행, 후행 상관관계도 랜덤성으로 있음. 지역별 연관성을 찾기 힘듬
  ⇒ 결측치 처리를 위한 다른 변수 활용 근거 희박

## 모델링

#### 제약사항
- 결측 데이터 비중 및 시점이 다양함. 특히 학습 데이터 종료 시점의 결측치가 많은 경우 테스트 데이터를 예측하는데 학습 데이터를 활용하기 쉽지 않음
- 각 사업장별로 동일한 항목을 측정했지만 항목별로 연관성이 다르기 때문에 결측치 보정에 다른 변수를 활용하거나 다변량 예측에 활용하는 것은 적절한 방법이 아닌 것으로 판단됨
- 또한 사업장별로 다른 모델이나 방법론을 사용하게 되면 모델이 강건(robust)하지 못하고  경진대회의 모델링 시간 제한에 걸릴 확률이 높음
⇒ 데이터의 정합성, 유효성, 정보가 매우 중요한 딥러닝, 머신러닝 기법보다 통계적인 접근으로 문제를 해결하고자 함

#### 프로세스
시계열 데이터 특성을 파악하여 통계적으로 유의미한 정보를 예측에 활용
<img width="713" alt="image" src="https://user-images.githubusercontent.com/30097727/227517782-7d4fbbed-fb07-46fc-b376-55bfa42f003c.png">
- Trend 특성 : 시계열 데이터에 방향성이 있는지 확인하고 통계적으로 유의미한 경우 해당 기울기를 반영하여 예측에 반영
- Seasonal 특성 : 매 계절마다 동일한 패턴이 보이는 경우 이를 계절적인 특성으로 예측에 반영함
